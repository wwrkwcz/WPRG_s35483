Złożone Równania Kwadratowe

-½x2 + 4x + 10 = 0

Δ = b2 - 4ac

⇒ a = -½, b = 4, c = 10

⇒ Δ = 42 - 4(-½)(10)

⇒ Δ = 16 + 20 = 36

⇒ √Δ = √36 = 6

x1 = (-b - √Δ) / 2a

⇒ x1 = (-4 - 6) / 2(-½) = 10

x2 = (-b + √Δ) / 2a

⇒ x2 = (-4 + 6) / 2(-½) = -2

x1 = 10, x2 = -2
